# Instructions for Using Doc Search

1. **Generate Text Chunks:**
   - Navigate to the `web_scrape` directory.
   - Follow the instructions provided to generate the text chunks that will be used as input for the Raptor model.

2. **Install the Raptor Package:**
   - Install the necessary package for creating the Raptor tree.
   - Follow the installation instructions at [Raptor GitHub Repository](https://github.com/parthsarthi03/raptor).
